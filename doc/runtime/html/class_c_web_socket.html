Detailed Description

Function pointer to register the routes.

WebSocket endpoint that queues messages and dispatches JSON commands.

Incoming multi-frame messages are captured as CWebSocketMessage objects, queued, and later dispatched from the application loop. Selected commands can require authentication before they are processed.
