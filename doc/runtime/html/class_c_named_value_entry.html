Detailed Description

template<typename TValue>
class CNamedValueEntry< TValue >

Single key/value entry used by CNamedValueTable.

The entry owns a copy of the key string. The value is stored directly and is intentionally public so callers can update it after a lookup without another wrapper function.

Template Parameters
TValueType of the stored value. The value type must be copyable.
