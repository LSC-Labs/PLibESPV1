Detailed Description

Simple delay class to avoid DOS calls to functions.

Should avoid to cal services too often (e.g. web requests).

Usage: CDelay oDelay(5000); // 5 seconds delay if(oDelay.isDone()) { ... ; oDelay.restart(); }

As an alternative, you can use start(nDelayMillis) to set a new delay time. Use reset() to stop the delay.

Parameters
